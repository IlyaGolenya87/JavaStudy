1. Посмотреть какие ветки есть git branch
* обозначает в какой ветке я нахожусь

2. git branch имя ветки - создаст ветку с новым названием

3. Переключение на созданную ветку:
git checkout имя ветки

4. Создание проекта и сохранение его в папку с проектом

5. Коммитим

6. git push origin имя ветки в которую пушим (должна быть моя ветка)

ДОМА:

1. Скопировать репозиторий
2. запустить терминал из-под папки с репозиторием
3. git fetch --all (скачивает список веток с удаленного репозитория на локальный)
4. показать список веток git branch -a (или без -а)
5. Переключение на свою ветку: git checkout имя ветки
6. git pull origin master
